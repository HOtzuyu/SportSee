## Installation

---

For launch the project in local.

>

```
$ git clone git@github.com:HOtzuyu/sportsee.git
$ npm install
$ npm run sass
$ npm start
```

Side information: Clone, install and run `the back-end` (with yarn) before start the project !

You can generate jsdoc after cloning.

```
npm run doc
```
